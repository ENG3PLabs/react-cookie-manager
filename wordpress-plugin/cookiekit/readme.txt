Professional GDPR & CCPA compliant cookie consent solution with beautiful design, automatic script blocking, and complete cookie compliance.

== Description ==

CookieKit is a professional-grade cookie consent solution that helps you comply with GDPR, CCPA, and other privacy regulations. With its beautiful design and powerful features, it makes cookie compliance simple and user-friendly.

= Features =

* Beautiful, responsive cookie consent banner
* Automatic script blocking for non-consented categories
* GDPR & CCPA compliant
* Multiple design options (banner, popup, modal)
* Light and dark themes
* Customizable text and translations
* Automatic consent logging (2,000 logs/month free)
* Easy to set up and configure

= Premium Features (via CookieKit.io) =

* Advanced consent logging and analytics
* Geographic targeting
* Custom branding
* Advanced script blocking
* Priority support

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/cookiekit`, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > CookieKit to configure the plugin
4. Sign up at CookieKit.io for a free account to enable consent logging
5. Copy your CookieKit ID from CookieKit.io and paste it in the plugin settings

== Frequently Asked Questions ==

= Is this plugin free? =

Yes! The plugin is completely free and includes 2,000 consent logs per month. For additional features and higher log volumes, premium plans are available at CookieKit.io.

= Is this plugin GDPR compliant? =

Yes, CookieKit is built with GDPR compliance in mind. It includes all necessary features for proper cookie consent management.
